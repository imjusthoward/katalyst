# Tutoring Deployment Requirements

## Users

- tutors
- students
- operations leads

## Core jobs

- track tutoring sessions
- see student and tutor status at a glance
- access standard resources and notes quickly
- capture operational updates
- preserve continuity when roles rotate

## Minimum MVP

- authenticated dashboard
- people directory
- tutoring session list
- announcements
- resource library
- ops notes
- simple status dashboard

## Guardrails

- internal-first
- keep permissions simple at the start
- prefer clear operational views over dense feature sets
